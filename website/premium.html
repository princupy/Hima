Redeem Flow

Token workflow in 4 steps

Keep sales and provisioning controlled: token generated by owner, redeemed by guild admin only.

Step 1Owner runs premiumtoken and chooses plan duration.
Step 2Bot generates secure token mapped to duration + price metadata.
Step 3Guild admin runs premiumredeem <token> in target server.
Step 4Guild premium, theme controls and 24/7 access go live instantly.
